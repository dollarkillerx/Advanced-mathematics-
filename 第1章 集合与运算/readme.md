集合与运算
===

### 集合的概念
集合：具有某种特定性质的事物的总体。
组成这个集合的事物称为该集合的元素
a ∈ M a 属于 M
a ∉ M a 不属于 M

``` 
a = {a,b,c,d,x}      列举法
M = {x|x所具有性质P}   描述法

B = {x|x ∈ R,x^2 - 1 = 0}   => b={-1,1}
```

#### 集合相等
若a ⊆ b,且 b ⊆ a ，就称集合a 与 b相等
例: A = {1,2} , B = {x|x^2 - 3x + 2 = 0},则A = B

空集： 不包含任何元素称为空集 Ø (空集是任何元素的子集)
